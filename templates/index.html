Output Format

The forecast CSV includes:

  • as_of - Forecast origin date
  • h - Forecast horizon (periods ahead)
  • Date - Forecast target date
  • actual - Actual value (if available)
  • mean - Point forecast
  • Confidence intervals - 95%, 90%, 80%, 60% bounds
  • type - "backtest" or "future_forecast"
